X415.2 - Introduction to Data Science

Introduction to Data Science

Course Description

This introductory course in data science will help you understand the core concepts of data science and predictive modeling illustrated through the use of the R language. This course covers some of the fundamentals of effectively executing data science projects including data gathering and cleaning, exploratory data analysis, data visualization, predictive modeling, collaboration and communication.

Course Learning Objectives

R environment Basic R programming with the tidyverse Exploratory data analysis Avoiding common data science pitfalls Linear and nonlinear predictive modeling Regression and classification Assessing model accuracy Reproducible computation with R Markdown and git Collaborate and communicate results effectively

Intended Audience

If you want a modern and practical introduction to the foundations of data science and predictive modeling using the R programming language, then this course is for you.

Prerequisites

The only prerequisite is an introductory statistics course. It will of course help if you have some programming or math background, but this course is taught at a high level.

Methods of Instruction

Lectures In-class examples In-class exercises (group and individual) Programming exercises Data analysis project (group or individual)
